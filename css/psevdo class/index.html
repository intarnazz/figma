Псевдоклассы гиперссылок

:link Применяется к ссылкам, которые ещё не посещались пользователем, и задаёт для них стилевое оформление 
    a:link{
    color:red;
    }

:visited Применяется к ссылкам, уже посещённым пользователем, и задаёт для них стилевое оформление 
    a:visited{
    color:green;
    }

Посетите эту ссылку и увидите, как она становится зеленой.

Псевдоклассы пользовательских действий

:hover Определяет стиль элемента при наведении на него курсора мыши, но при этом элемент еще не активирован, иными словами кнопка мыши не нажата. 
    .userActions:hover{
    background:red;
    box-shadow: 2px 3px 5px #000;
    color: #fff;

Наведите курсор на эту ссылку и увидите, как она становится красной.

:active Определяет стиль активного элемента. Это такое состояние элемента, которое происходит между щелчком и отпусканием клавиши мыши. В основном применяется к ссылкам и кнопкам, но ими не ограничен. 
    .userActions:active{
    background:black;
    }
    a:active {
    background:black;
    color: #fff;
    }

Кликните мышкой на эту ссылку и увидите, как она становится черной.

:focus Определяет стиль для элемента, получающего фокус. Например, им может быть текстовое поле формы, в которое устанавливается курсор. 
    .userAct:focus{
    background:yellow;
    border: 5px solid red;
    }
